# C-2.DONEM
Bu depo, C dili kullanarak veri yapıları, dosya işlemleri ve ileri seviye sıralama algoritmaları üzerine yaptığım çalışmaları içermektedir.


Programlama Dili: C

Temel Konular: * Bağlı Listeler (Linked Lists): Tek yönlü ve çift yönlü bağlı listeler, araya ekleme ve silme işlemleri.

Yığın (Stack): Bağlı liste tabanlı yığın yapısı.

Sıralama Algoritmaları: Quick Sort, Merge Sort, Selection Sort ve Insertion Sort.

Bit Düzeyinde İşlemler: Bit kaydırma (shift) ve bit değiştirme işlemleri.

Dosya Yönetimi: Dosyadan veri okuma ve işleme.

Struct Yapıları: Öğrenci otomasyonu ve vize-final hesaplamaları.
